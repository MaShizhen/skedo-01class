# Skedo 

Skedo——旨在减少产品、技术、UI间沟通的低代码搭建平台。

## 开发环境搭建

1. node >= 12 （官网下载）

2. ts-node `npm install ts-node -g` 

3. yarn： `npm install yarn -g` 

4. 安装

   ```shell
   yarn install
   
   # 安装所有:
   #  依赖
   #  link
   npm run reinstall
   
   # 编译TS模块
   npm run build-ts
   
   ```

   
5. 启动： `npm start`
